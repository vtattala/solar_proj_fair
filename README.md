

This is the public repository for the coding done for the project **HelioAegis** for the **Greater San Diego Science and Engineering Fair**. All notebooks were coded in **Jupyter Notebook** using **Python** and are stored here.

All plots for **Phase 2** are specifically stored as individual files due to GitHub requirements. The outputs from the Phase 2 notebook had to be wiped, so the plots are uploaded individually, along with the 3D trajectory plots. However, **Phase 1** has all plots included as part of the outputs in the Phase 1 notebooks, so they do not need to be uploaded individually.

This repository is the source of all the rigorous work completed over the last six months to develop this project.
